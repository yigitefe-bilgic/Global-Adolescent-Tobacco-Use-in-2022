# Global Adolescent Tobacco Analysis (2022)
This project provides a comprehensive analysis of tobacco prevalence among adolescents (13-15) using **UNICEF** data.

## Key Features
* **Data Engine:** Powered by **Polars** for high-performance data manipulation.
* **Visualizations:** Interactive maps and charts using **Plotly**.
* **Design:** Professional HTML/CSS integrated Google Colab interface.

## Research Context
Conducted as part of the MSc Management program at **Dublin City University (DCU)**. This study focuses on identifying high-risk regional clusters and the narrowing gender gap in nicotine dependency.
